# @react-native-ui-components/toolbar

Toolbar component for React Native

## Installation

```sh
npm install @react-native-ui-components/toolbar
```

### peer dependencies:

-   react-native-safe-area-context

## Usage

```js
import {Toolbar} from "@react-native-ui-components/toolbar";

// ...

<Toolbar></Toolbar>;
```
